# PawHub App - React Native Social Media for Pets

## Overview

Welcome to PawHub, the mobile app version of our full-stack social media project for pets, built using React Native. This app allows pet owners to connect, share posts, and showcase their furry friends in a dedicated mobile community.

## Features

- **User Authentication:** Secure user authentication using MongoDB for user storage. Users can sign up, log in, and manage their profiles.
- **Post Creation and Sharing:** Users can create posts and share images to engage with the community.
- **Feed:** A dynamic feed displays posts from users that others are following, creating an interactive and engaging experience.
- **User Profiles:** Every user has a profile page where they can view and edit their personal information, as well as see their posts and followers.
- **Responsive Design:** PawHub is designed to provide a seamless experience on mobile devices.

## Tech Stack

- **Mobile App:** React Native
- **Frontend:** Expo Client
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Hosting:** Expo for app deployment

## Getting Started

### Prerequisites

- Node.js installed on your machine
- MongoDB instance (local or remote)
- Expo CLI installed globally: `npm install -g expo-cli`

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/rundex0/Large-Project-App.git
   cd pawhub-app
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Configuration:

   - Create a `.env` file in the project root and configure the following:

     ```env
     MONGODB_URI=your_mongodb_connection_string
     SECRET_KEY=your_jwt_secret_key
     ```

     Replace `your_mongodb_connection_string` with your MongoDB connection string and `your_jwt_secret_key` with a secret key for JWT.

4. Run the application:

   ```bash
   npm start
   ```

   Follow the instructions to open the app on your preferred device using the Expo Client.
